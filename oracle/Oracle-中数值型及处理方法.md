# 数值型

```
number[(precision [, scale])]
```
- `precision`, 数值精度
- `scale`,小数点后位数

# 数值处理

- `abs()`
- `round()`
- `ceil()`
- `floor()`
- `mod()`
- `sign()`
- `sqrt()`
- `power()`
- `trunc()`
- `chr()`
- `to_char()`
